# 针灸大原则
针是针，灸是灸。同一个部位不能同时下针和灸。
## 针
针可以用来治所有的病，下针的时候左病右治，右病左治。
## 几种情况下不用针
1. 气血大虚后
2. 房事后
3. 喝酒后
## 灸
灸的目的是为了找水，灸主要用艾草。灸的单位是壮，灸单不灸双。
## 灸的操作方式
1. 隔姜灸。一片姜可以灸多次。
2. 隔盐巴灸。用青盐巴，炒红后凉温后灸
3. 隔蒜灸。用独蒜，比如治肺部问题。
## 几种情况下不灸
1. 多汗部位
2. 热症部位
3. 发烧时
4. 眼睛，阴部
## 艾草
1. 纯阳
2. 艾草性向下
3. 找水
# 怎么判断病进退趋势
1. 从四肢往身体中心时为病进
2. 从身体中心往四肢时为病退

# 针的急救方式
不论病因，大原则先把命救回来
1. 人中。两点分三段，在第二点往头的方向斜着下针。
2. 十宣放血。先手后脚，速度要快。
3. 涌泉穴
4. 百会穴。中风时，病人躺下，头下放接血的桶。

# 穴道的作用

# 季节和下针深浅的关系
春夏的时候下针浅，秋冬下针深。
# 病在哪里和下针深浅的关系
病在皮毛或肺部下针在皮毛；病在肌肉或脾下针在肌肉旁边；病在血脉或肝下针在血脉旁边；病在骨头或肾下针在骨头旁边

### 腹深似井，背薄似饼
下针要注意深浅，下针要小心。


# 实际下针具体位置的取舍
先大致定穴道的位置，在通过手感找到下针的具体点。这个点一定在肌肉，血脉，骨头之间。绝对不能下针到肌肉，血脉，骨头的上面。

## 成人与幼儿
1. 成人可以留针，幼儿不可以留针。按照14岁作为界限，14岁以下不可以留针，要单刺，14岁以上可以留针

## 晕针
1. 正常治疗如果出现晕针，也就是瞑眩感觉，是好的迹象。包括其他方式的治疗，比如吃药等，出现瞑眩感觉，是好的迹象。
2. 下针前病人要卧下，不要站着或坐着下针，否则会出现晕针的问题。

# 井荣俞经合
”病在藏者,取之井;病变于色者,取之荥;病时间时甚者,取之输,病变于音者,取之经;经满而血者,病在胃,及以饮食不节得病者,取之于合“
## 井
井穴通通在手脚指头的末梢。
病在脏取井。意思是病在越深的地方取井穴进行治疗。
## 荥
病变于色者,取之荥
## 俞
病时间时甚者，固定时间发病，参考地支，取之输
## 经
病变于音者,取之经
## 合
病在胃,及以饮食不节得病者,取之于合

### 独立丈量法
当同身寸不准的时候，通过选区两点，通过平均分配两点间的距离来定位穴道。
例子：神阙和曲骨之间的穴道的定位，将神阙和曲骨之间通过4个点平均分成5段。这样的每一段就是”一寸“，第一个点就是中极穴。中极上一寸是关元。关元上一寸是石门。石门上一寸是阴交。阴交下半寸是气海。
### 曲骨穴

### 中极穴

### 关元穴

### 绝育穴
关元和石门中间的奇穴，禁针灸。
### 石门穴

### 气海穴
石门和阴交中间的奇穴
### 阴交穴
也是三焦的募穴，治阴部出汗湿痒。
### 神阙穴
禁针。可灸。中风，肠鸣。

# 手太阴肺经
手太阴肺经有11个穴道。”中府云门天府诀，侠白尺泽孔最存，列缺经渠太渊涉，鱼际少商如韭叶“
### 中府穴
手太阴肺经的第一个穴道。肺的募穴。云门穴下1寸半（隔一个肋骨）。不直刺。久咳。
### 云门穴
锁骨下方的凹洞。肺的募穴。不直刺。久咳。
### 天府穴
手肘到腋下距离的中点。治流鼻血。
### 侠白穴
天府下1寸。辅助天府
### 尺泽穴
手肘处大筋的外侧。手太阴肺经的合穴。肺实泄之。手太阴肺经是一条阴经，合穴属水。肺属金，因此尺泽穴又是手太阴肺经的子穴。
### 孔最穴
尺泽下3寸。手太阴肺经的郗穴。消炎。痔疮。
### 经渠穴（关）
距离手腕1寸高骨下。手太阴肺经的经穴。属金。也是手太阴肺经的本穴。
### 列缺穴（尺）
经渠下5分。主任脉，身上的任脉是列缺在管。任脉上痛的时候可以下列缺穴。从喉咙到阴部这条线上的位置痛。
### 太渊穴（寸）
经渠上5分。脉会太渊。手太阴肺经的俞穴。属土。肺属金，因此太渊穴又是手太阴肺经的母穴。脉病治太渊。虚则补其母，肺的虚证可以补这个穴道。
### 鱼际穴
拇指根部到手腕之间的那根骨头的中间处，和手掌肌肉的中间。手太阴肺经的荣穴。乳房相关的疾病。
### 少商穴
拇指指甲盖根部处，骨头和肌肉之间。手太阴肺经的井穴。扁桃体发炎。退烧。

# 手少阳三焦经
三焦行阳。三焦分为上中下三焦。”决渎之官，水道出焉“
下焦：肚脐以下。如渎，
中焦：肚脐到胸壁骨之间。如沤，
上焦：胸壁骨以上。如雾，轻，干净。


# 虚

# 实
有东西；拒按；要泄；逆着气扎叫泄；新病为实；绷紧；
# 虚中带实
酸多痛少。先补后泄
# 实中带虚
痛多酸少。先泄后补

# ”血旺气衰，气旺穴衰“

# 下针深浅的问题
1. 根据病人的症状来。如果是皮肤病，浅。如果是水病，深。如果病在骨头，要下到骨头旁边。
2. 如果病在脏，深。如果病在腑，浅。

# 膀胱中小便的来源

2. 来自三焦的水，三焦和膀胱是连着的。
